- Avatar circle
- Name, title, and location
- 3 stat boxes (followers, posts, likes)
- Bio section
- Follow button

**Requirements:**
- Use proper spacing (padding and margin)
- Use appropriate text sizes
- Add borders and rounded corners
- Use shadow for depth
